# Vinicius Barros | Mobile Developer Portfolio

Portfolio pessoal criado para apresentar minha trajetória como desenvolvedor mobile e front-end. Este site serve como porta de entrada para minha carreira, showcasing projetos, experiências e habilidades.

## Stack

- **React** + **Vite** — Interface dinâmica
- **Tailwind CSS** — Estilização
- **Framer Motion** — Animações

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Estrutura

```
src/
├── components/    # Componentes React
├── data/          # Dados do perfil
└── App.jsx        # Componente principal
```

## Deploy

Build gerado em `dist/`. Deploy em qualquer hospedagem estática (Vercel, Netlify, GitHub Pages).
